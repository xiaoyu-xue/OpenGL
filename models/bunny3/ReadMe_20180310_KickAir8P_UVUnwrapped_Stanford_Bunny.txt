ReadMe_20180310_KickAir8P_UVUnwrapped_Stanford_Bunny.txt
--------------------------------------------------------

The blend ( https://www.blender.org/ ) files include a modified (patched, bottom removed, UV-unwrapped & textured) version of the Stanford Bunny in a simple scene.  These are set up to use Troy Sobotka's Filmic OpenColorIO configuration ( https://github.com/sobotka/filmic-blender ).

The xcf ( https://www.gimp.org/ ) files are the textures' layers for easy modification.  The original grayscale maps were made by baking Joakim Tornhill's "pointiness" procedural shader setup ( https://www.blendernation.com/2018/03/05/procedural-wear-tear-using-cycles-nodes/ ).
